# DeepSeek-streamlit 高拟人感聊天机器人
基于DeepSeek大模型 + Streamlit网页端实现的人格化聊天AI

## ✨ 功能全部实现
1. 网页版聊天UI界面
2. DeepSeek官方API流式对话
3. 超强上下文长记忆，不会遗忘对话内容
4. 自定义人设背景，固定角色性格对话
5. 极低AI感、真人口语化聊天
6. 一键清空对话重新开始

## 🛠️ 运行方法
1. 填入自己的DeepSeek API Key
2. 安装依赖
```pip install -r requirements.txt```
3. 启动项目
```streamlit run chat_app.py```
